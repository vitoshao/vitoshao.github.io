要建立 Android 開發環境，會用到以下幾個工具軟體：

- [Eclipse](https://www.eclipse.org/downloads/)：一套用來撰寫 Java 程式語言的開發工具。
- [Android SDK](http://developer.android.com/sdk/index.html)：提供開發 Android 應用程式所需要的 API 類別庫。
- [Android Development Tools (ADT)](http://developer.android.com/tools/sdk/eclipse-adt.html)：ADT 是安裝在 Eclipse 中的 plugin 套件，它提供一個強大的整合環境，讓你在 Eclipse 中可以更快速的開發 Android 應用程式。
- [ADT Bundle](http://developer.android.com/sdk/index.html)：ADT Bundle 是 Google 為了降低 Android 入門門檻所包裝而成的開發工具，整合了Android SDK、eclipse、Android Development Tools (ADT)，而且只要解壓縮後就可以使用了

# 安裝 Genymotion 模擬器

Genymotion 模擬器有提供免費版本可下載使用，但必須先申請成為會員才可下載，且執行模擬器時，會用到這個認證。  

另外，模擬器使用 VirtualBox 來啟動一個虛擬機器，以建立手機或平板的模擬環境，所以在模擬器安裝過程中，也會同時安裝 VirtualBox 。  如果電腦中早先已安裝過 VirtualBox ，則可以跳過這些步驟，或者選擇不含 VirtualBox 的安裝檔下載。  

## 安裝與設定

在模擬器安裝完之後的首次執行時，系統會要求你設定「模擬設備」，你也可以自行增加多個模擬設備，每個模擬設備就像一支手機或平板一樣。  

不過要設定模擬設備前，必須先取得相關設備的目錄，所以首次執行 Genymotion 模擬器時，必須由 Connect 功能，登入 Genymotion 已取得相關設備的目錄。  

模擬設備目錄中包含多款手機型號，你可依需要，自行選定要使用的設備，系統就會開始下載模擬器的檔案。
