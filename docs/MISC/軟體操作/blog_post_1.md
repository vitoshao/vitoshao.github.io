雖然我也很少用這軟體, 但是剛在網路上看到這篇內容, 就順便記錄下來  
其實平常只會用到文字的左移右移罷了, 原來這叫 Indent / Outdent Code  
Hotkey :  Ctrl-Shift-&gt;
