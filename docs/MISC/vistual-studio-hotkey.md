# VS 快捷鍵大全

Visual Studio 提供了大量的快捷鍵來提升開發效率。以下是一些常用的快捷鍵列表：

## 註解操作
- `Ctrl + K, C`：註解選中的程式碼行。
- `Ctrl + K, U`：取消註解選中的程式碼行。

## 行操作
- `Alt + Up/Down`：向上或向下移動當前行。
- `Ctrl + L`：刪除當前行。
- `Ctrl + D`：複製當前行並粘貼在下一行。 若有選中內容，則複製選中的內容並粘貼在右側。
- `Ctrl + Enter`：游標所在行數插入一行空行。
- 
## 段落操作

- `Ctrl + M, O`：折疊所有區塊
- `Ctrl + M, L`：打開所有折疊區塊
- `Ctrl + M, M`：打開/折疊游標所在的區塊
- `Ctrl + M, A`：整個文件折疊起來
- `Ctrl + K, D`：格式化文件

## Code Cleanup 操作

- `Ctrl + K, E`：移除沒有使用到的 using 指令。
- `Ctrl + R, G`：移除沒有使用到的 using 指令, 並重新排序。
- `Ctrl + .`：快速啟動控制項目及重構
- `Ctrl + ←→`：一次跳過一個單字
- `Ctrl + Shift + ←→`：一次選取一個單字

## 視窗操作

- `Ctrl + F4`：關閉目前文件分頁
