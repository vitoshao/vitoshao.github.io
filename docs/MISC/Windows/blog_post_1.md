如果想把自已的 NB 充當無線基地台，可以參照以下步驟。  在設定前，請先確認該筆電的 wifi 已經開啟。  以下示範環境：Win7。  

### 1. 設定虛擬網路

在命令輸入視窗中，輸入以下指令，用來設定一個虛擬網路。
netsh wlan set hostednetwork mode=allow ssid=你要的虛擬網路名稱 key=密碼  
這時應該會出現以下回應訊息：

![](http://127.0.0.1:8080/_images/public/nb-wap-01.png)

### 2. 啟動虛擬網路

接著再使用以下命令，啟動這個虛擬網路。
netsh wlan start hostednetwork  
如果啟動成功，則會出現以下回應訊息：

![](http://127.0.0.1:8080/_images/public/nb-wap-02.png)

如果出現以下回應訊息，那麼表示啟動失敗，請參考下一步驟設定。

![](http://127.0.0.1:8080/_images/public/nb-wap-03.png)

### 3. 啟用 Microsoft 主控虛擬網路介面卡

這個步驟是要確認筆電的**虛擬網路介面卡**是否已啟用。如果上一步驟已啟動成功，則可以跳過這個步驟。

3.1 請[開啟裝置管理員](http://windows.microsoft.com/zh-tw/windows/open-device-manager#1TC=windows-7)，找到「Microsoft 主控虛擬網路介面卡」。

![](http://127.0.0.1:8080/_images/public/nb-wap-04.png)

如果找不到這個項目，可以點選[檢視]功能表中，執行[顯示隱藏裝置]。

![](http://127.0.0.1:8080/_images/public/nb-wap-05.png)

3.2 如果這個項目前方有個**向下箭號**表示虛擬網路介面卡尚未啟用，如下圖：

![](http://127.0.0.1:8080/_images/public/nb-wap-08.png)

3.3 你可以在右鍵選單中，執行[啟用].

![](http://127.0.0.1:8080/_images/public/nb-wap-06.png)

3.4 虛擬網路介面卡啟用後的狀態如下圖：

![](http://127.0.0.1:8080/_images/public/nb-wap-07.png)

### 4. 檢查設定的虛擬網路網路

如果設定無誤，你應該可以在「檢查作用中的網路」，看到剛才建立的無線虛擬網路名稱。  

![](http://127.0.0.1:8080/_images/public/nb-wap-10.png)

### 設定「網路共用」

接下來必須設定「網路共用」，意思是說，目前真正可以連線上網那個網路連線，分享給我們在前面步驟所設定的那個無線虛擬網路共用。  像下圖中，  那個「區域連線」是目前對外的真正網路連線，  而「無線網路連線２」則是自訂的虛擬網路。  我們必須將這個「區域連線」分享給「無線網路連線２」，所以點繫一下「區域連線」，開啟設定。  

![](http://127.0.0.1:8080/_images/public/nb-wap-14.png)

按下「內容」。

![](http://127.0.0.1:8080/_images/public/nb-wap-12.png)

在「共用」頁框裡，把「允許其它網路使用者透過這台電腦的網際網路連線來連線」打勾。  並在「家用網路連線」選項中，選擇自訂的虛擬網路連線，本例為「無線網路連線２」。  也就是允許「無線網路連線２」透過「區域連線」來連線。  

![](http://127.0.0.1:8080/_images/public/nb-wap-13.png)

PS.  如果「允許其它網路使用者透過這台電腦的網際網路連線來連線」的勾原本就已經打勾，  而「家用網路連線」的值又不等於你的虛擬網路連線設定，且值又無法變更，  那麼請先取消這個勾選，按確定離開後，再進來一次就可以設定了。

### 5. 使用手機連線上網

最後你就可以在手機中找到這個虛擬無線AP，正確設定密碼後，就可以連線上網了。
